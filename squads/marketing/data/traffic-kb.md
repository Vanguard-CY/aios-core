# Base de Conhecimento — Time de Tráfego

## Plataformas Principais

### Meta Ads
- Funil recomendado: TOF (broad/lookalike) → MOF (engajamento) → BOF (retargeting)
- Budget mínimo para teste: R$ 50/dia por conjunto de anúncios
- Testar 3-5 criativos por conjunto antes de julgar performance
- Frequência ideal: 2-4 para BOF, 1-2 para TOF
- ROAS meta: depende da margem (margem 40% = ROAS mínimo 2.5)

### Google Ads
- Search: captura demanda existente — ideal para BOF
- Performance Max: usar depois de ter histórico de conversão
- Display: reconhecimento e retargeting
- YouTube: awareness e lançamentos

## Fórmulas de Performance

### Cálculo de ROAS Break-Even
```
ROAS Break-Even = 1 / Margem de Contribuição
Exemplo: Margem 40% → ROAS mínimo = 2.5
```

### Cálculo de CPA Máximo
```
CPA Máximo = Ticket Médio × Margem
Exemplo: Ticket R$500, Margem 40% → CPA máximo = R$200
```

## Regras de Otimização

1. **Não otimize antes de ter dados suficientes** — mínimo 50 conversões por conjunto
2. **Escale com cautela** — aumento de budget máx 20-30% a cada 3-4 dias
3. **Matou um anúncio?** — documente o motivo para aprender
4. **Criativo cansado** — CTR cai mais de 50% do pico? Renove
5. **NEVER** otimize custo quando campanha está em aprendizado

## Glossário
- **ROAS**: Return on Ad Spend (Receita ÷ Gasto)
- **CPA**: Custo Por Aquisição
- **CPM**: Custo Por Mil impressões
- **CTR**: Click-Through Rate (Cliques ÷ Impressões)
- **Frequency**: Quantas vezes a mesma pessoa viu o anúncio
- **LTV**: Lifetime Value do cliente
