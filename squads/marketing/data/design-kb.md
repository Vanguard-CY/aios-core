# Base de Conhecimento — Time de Design

## Princípios de Criativo que Converte

### Os 3 Segundos Decisivos
O criativo tem 3 segundos para parar o scroll. Elementos que param o scroll:
- Rosto humano olhando para o texto ou câmera
- Contraste de cor forte
- Movimento inesperado (no vídeo)
- Texto bold e grande no primeiro frame
- Situação de conflito/problema reconhecível

### Hierarquia Visual
```
1º - Hook visual (o que para o olho primeiro)
2º - Contexto (onde estou? o que é isso?)
3º - Benefício (o que ganho?)
4º - CTA (o que fazer agora?)
```

## Estratégia de UGC

### Quando usar UGC vs Produção
- **UGC**: Mais barato, conversão muitas vezes melhor (autenticidade)
- **Produção**: Branding, lançamentos, awareness de marca

### Brief de UGC Eficiente
1. **Problema a resolver** — o criador deve demonstrar compreensão
2. **Produto em uso** — mostrar, não só falar
3. **Resultado específico** — número, antes/depois, transformação
4. **Duração e formato** — 15-30s para Stories, 30-60s para Feed
5. **Tom** — autêntico, não roteirizado demais

## Erros Mais Comuns em Criativos

- ❌ Logo muito grande no início (ego antes de valor)
- ❌ Texto pequeno demais (mobile first!)
- ❌ Muita informação num único criativo
- ❌ CTA vaga: "Saiba mais" vs "Compre agora com 20% off"
- ❌ Não testar variações (cor, rosto, ângulo, texto)
