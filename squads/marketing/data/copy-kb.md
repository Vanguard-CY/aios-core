# Base de Conhecimento — Time de Copy

## Frameworks de Copywriting

### PAS (Problema → Agitação → Solução)
Melhor para: anúncios frios, copy de consciência de problema

```
PROBLEMA: Identifique a dor específica
AGITAÇÃO: Aprofunde a dor, mostre as consequências
SOLUÇÃO: Apresente o produto como a saída
```

### AIDA (Atenção → Interesse → Desejo → Ação)
Melhor para: landing pages, emails, VSL

### Hook Formula
```
[Número/Resultado] + [Benefício] + [Tempo] + [Sem objeção]
Exemplo: "Aprenda a dobrar seu ROAS em 7 dias sem aumentar budget"
```

## Estrutura de VSL

1. **Hook** (0-30s): Para o scroll, cria curiosidade irresistível
2. **Agitação** (30s-2min): Aprofunda a dor, cria urgência
3. **Mecanismo Único** (2-4min): O que torna sua solução diferente
4. **Prova Social** (4-6min): Depoimentos, resultados, dados
5. **Oferta** (6-8min): O que vai levar, por qual preço, quando
6. **CTA** (8min+): Chamada clara e única para ação

## Objeções Universais

| Objeção | Como destruir |
|---------|--------------|
| "É muito caro" | Foque no ROI, compare com o custo de NÃO resolver |
| "Não tenho tempo" | Mostre quanto tempo economiza |
| "Não vai funcionar pra mim" | Prova social específica do perfil deles |
| "Vou pensar" | Urgência + escassez genuína |
| "Não confio" | Garantia + prova social + transparência |

## Princípios de Copy de Alto Impacto

1. **Escreva para uma pessoa** — nunca para uma massa
2. **Seja específico** — "dobrou o faturamento em 47 dias" vs "cresceu muito"
3. **Benefício antes de feature** — o que muda na vida deles, não o que o produto faz
4. **Prova social específica** — nome + contexto + resultado mensurável
5. **CTA único e claro** — uma só ação por peça de copy
