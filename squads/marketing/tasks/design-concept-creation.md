# Task: Criação de Conceito de Campanha

## Objetivo
Desenvolver o conceito criativo completo de uma campanha — a ideia central que guiará todos os criativos e será executada pelo `@visual-designer` e `@video-editor`.

## Inputs Necessários
- Objetivo da campanha (conversão, awareness, lançamento)
- Produto/serviço a promover
- Público-alvo (dados demográficos + psicográficos)
- Contexto da campanha (é lançamento? promoção? branding?)
- Tom desejado (aspiracional, urgência, educativo, divertido)
- Referências ou inspirações (opcional)

## Passo a Passo

### STEP 1: Imersão no Público
Antes de criar qualquer coisa:
- **Quem é a pessoa que vai ver esse criativo?** (visualizar especificamente)
- **O que ela está sentindo quando rola o feed?** (contexto mental)
- **Qual é a dor/desejo que o produto resolve?**
- **O que faria essa pessoa parar o scroll imediatamente?**

### STEP 2: Definir a Big Idea
A Big Idea é o conceito central que unifica toda a campanha:

```
Big Idea = [Emoção] + [Transformação] + [Mecanismo Único]

Exemplo:
"Antes você precisava de uma agência inteira para ter marketing de alto nível.
Agora você tem um time de IA que trabalha 24h por uma fração do preço."
```

A Big Idea deve:
- Ser comunicável em 1 frase
- Gerar curiosidade OU ressonância imediata
- Diferenciar da concorrência

### STEP 3: Desenvolver os Ângulos Criativos
Criar 3 ângulos diferentes para testar:

**Ângulo 1: DOR** — Foca no problema atual
```
Hook: [O que dói hoje para o público-alvo]
Corpo: Aprofunda a dor sem solução por um momento
Virada: Apresenta o produto como alívio
```

**Ângulo 2: ASPIRAÇÃO** — Foca no futuro desejado
```
Hook: [Mostra o estado desejado já conquistado por alguém]
Corpo: Mostra o caminho de transformação
Virada: O produto é o veículo da transformação
```

**Ângulo 3: CURIOSIDADE** — Intriga sem revelar imediatamente
```
Hook: [Pergunta ou afirmação contraintuitiva]
Corpo: Desenvolve a tensão da curiosidade
Virada: Revela e conecta ao produto
```

### STEP 4: Definir Direção Visual para Cada Ângulo
Para cada ângulo, especificar:
- **Paleta de emoção:** Warm (calor, conforto) / Cool (profissional, tech) / Neutral
- **Estilo visual:** Realista/UGC / Mocionado / Ilustração / Produto em Destaque
- **Personagem/cenário:** Quem aparece? Em qual contexto?
- **Hook visual:** O que para o olho no primeiro frame?

### STEP 5: Pacote de Criativos por Ângulo

Para cada ângulo, definir a lista de peças:
```
ÂNGULO 1 — [Nome do Ângulo]:
  Vídeo principal: [duração] [formato] → @video-editor
  Static Feed: 1:1 → @visual-designer
  Stories: 9:16 → @visual-designer
  Carrossel (5 slides): → @visual-designer
```

### STEP 6: Linha Editorial da Campanha
Definir os elementos que devem ser consistentes em TODOS os criativos:
- Logo: posição e tamanho
- Paleta de cores obrigatória
- Tipografia
- CTA padrão
- Elementos de marca

### STEP 7: Briefing para os Times
Gerar:
- Brief para `@visual-designer` via `*brief-producao`
- Brief para `@video-editor` via `*brief-edicao`

## Output Esperado
Documento de conceito com: Big Idea, 3 ângulos criativos, direção visual por ângulo, lista de criativos a produzir e briefings prontos para os produtores.
